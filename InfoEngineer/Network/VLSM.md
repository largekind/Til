# VLSM : Variable Length Subnet Masking

サブネットで必要となるIPアドレス数に基づいて、サブネットごとに異なるサブネットマスクを使用する手法

最後のIPアドレスの先頭から1続きで始まるアドレスが対応する

例としては以下の表となる

| 10進数 |  2進数  | 16進数 |
|-------|----------|------|
| 128   | 10000000 | 80   |
| 192   | 11000000 | C0   |
| 224   | 11100000 | E0   |
| 240   | 11110000 | F0   |
| 248   | 11111000 | F8   |
| 252   | 11111100 | FC   |
| 254   | 11111110 | FE   |
| 255   | 11111111 | FF   |
