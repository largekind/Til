# データベースのキー一覧

## candidate key : 候補キー

表のある行を一意に特定することができる属性、または属性の組合せで主キーの候補となるキー
これにNOT NULL制約を加えると主キーになる

## primary key : 主キー

表のある行を一意に特定することができるの識別子として利用するのにもっとも好ましいものとして、
テーブル毎に設計者により選択・定義された、ただ一つの候補キー

## alternate key : 代理キー

候補キーのうち、主キーに選ばれなかったキー

## foreign key : 外部キー

異なる表の列と関連付けされるキー
