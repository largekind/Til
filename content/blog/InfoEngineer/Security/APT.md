# APT : Advanced Persistent Threats

Advanced : 高度
Persistent : per- 初めから終わりまで + sisto 立ち続ける = がんごな、持続的な、固執
Threat : 脅威 intrude(侵入)とかと同じ語源 thrに押し出すという意味がある

持続的標的型攻撃と呼ばれる。

特定組織を標的に複数の手法を組み合わせて気付かれないよう執拗(よう)に攻撃を繰り返すこと
