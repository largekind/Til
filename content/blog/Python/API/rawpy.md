# Rawpy

## 概要

RAW画像に対する読み込みや処理を行うためのライブラリ群

もともとlibrawと呼ばれるRAW画像のデコーダーをPythonで利用できるようにラップしたライブラリの模様

rawpyを使用すると、RAW画像ファイルを読み込み、色や露出の補正を行ったり、
画像を他のフォーマット（例：JPEG、PNG、TIFFなど）に変換したりすることができる
