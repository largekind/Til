# データカタログ

## 概要

データカタログは、企業や組織が所有するデータ資産を効果的に管理し、利用するためのツール。

データの発見、理解、アクセスが容易になり、データガバナンスの強化とデータ駆動型の意思決定が支援するのに利用。

主にメタデータ管理、データリネージ（データの流れと関係性の追跡）、アクセス管理などの機能を提供している。

## ツール一覧

以下のツールが、データカタログツールとして広く利用

- Apache Atlas
- Amundsen
- CKAN
- Metabase
- DataHub
- OpenMetadata
