## 論理

論理が一部違う。'=='が**型を考えない一致確認**なので、以下のようなパターンもtrueにしてしまう

基本的にはcの==と同じような、**型も含めた一致かどうかを確認**する使い方をする場合 **'==='を使う**

``` javascript
1 == '1' //文字列と数値 ==ではtrueになってしまう
1 === '1' //こちらはfalseになる
```
